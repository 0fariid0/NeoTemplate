# 3x-ui Subscription Theme Pack

نسخه‌ی تمیز و جمع‌وجور پروژه برای قالب‌های سابسکریپشن 3x-ui.

## نصب مستقیم

```bash
bash <(curl -Ls https://raw.githubusercontent.com/0fariid0/NeoTemplate/main/install.sh)
```

## دستورهای سریع

```bash
neotemplate              # باز کردن منو
neotemplate install neo-default
neotemplate install-all
neotemplate upgrade      # آپدیت تم‌های نصب‌شده
neotemplate update       # آپدیت منیجر و سپس تم‌ها
neotemplate list
```

## نکته

بعد از نصب هر تم، مسیر نمایش داده شده را در 3x-ui وارد کن:

`Panel Settings -> Subscription -> Sub Theme Directory`

مثال مسیر:

`/etc/3x-ui/sub_templates/neo-default/`
